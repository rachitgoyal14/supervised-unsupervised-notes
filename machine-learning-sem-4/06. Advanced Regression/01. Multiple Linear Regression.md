## General Formula

For **n independent variables**:
```
y = β₀ + β₁x₁ + β₂x₂ + β₃x₃ + ... + βₙxₙ + ε
```

Where:
- **y** = dependent variable (target/output)
- **β₀** = intercept (bias term)
- **β₁, β₂, ..., βₙ** = coefficients/weights for each feature
- **x₁, x₂, ..., xₙ** = independent variables (features)
- **ε** (epsilon) = error/noise term
- **n** = number of independent variables

---

## Number of Coefficients

**Total coefficients = n + 1**

- **n coefficients** (β₁, β₂, ..., βₙ) for independent variables
- **1 intercept** (β₀)

**Example:**
- 2 independent variables → 3 coefficients (β₀, β₁, β₂)
- 5 independent variables → 6 coefficients (β₀, β₁, β₂, β₃, β₄, β₅)

---

## Two Independent Variables Example
```
y = β₀ + β₁x₁ + β₂x₂ + ε
```

**Coefficients:** 3 total (β₀, β₁, β₂)

**Geometric Interpretation:** This represents a **plane** in 3D space
- x₁ axis
- x₂ axis  
- y axis
- The equation defines a flat plane

---

## Geometric Interpretation by Dimensions

| Independent Variables | Equation | Geometric Shape |
|----------------------|----------|-----------------|
| **1 variable** | y = β₀ + β₁x₁ + ε | **Line** (2D) |
| **2 variables** | y = β₀ + β₁x₁ + β₂x₂ + ε | **Plane** (3D) |
| **3+ variables** | y = β₀ + β₁x₁ + ... + βₙxₙ + ε | **Hyperplane** (n+1 dimensions) |

---

## Hyperplane Concept

When you have **more than 2 independent variables**, the regression model creates a **hyperplane**.

**Definition:** A hyperplane is the generalization of a plane to higher dimensions.

**Examples:**
- 3 independent variables → 4D hyperplane (impossible to visualize)
- 10 independent variables → 11D hyperplane
- 100 independent variables → 101D hyperplane

**Key Point:** We can't visualize hyperplanes, but the mathematics works the same way!

## Matrix Notation

For compact representation:
```
y = Xβ + ε
```

Where:
- **y** = (n × 1) vector of outputs
- **X** = (n × p) matrix of inputs (p = number of features)
- **β** = (p × 1) vector of coefficients
- **ε** = (n × 1) vector of errors

---

## Key Takeaways

1. **n independent variables** require **n + 1 coefficients** (including intercept)
2. **1 independent variable** → Line in 2D
3. **2 independent variables** → Plane in 3D  
4. **3+ independent variables** → Hyperplane in higher dimensions
5. **Noise term (ε)** accounts for unexplained variance
6. Same linear algebra principles apply regardless of dimensions
